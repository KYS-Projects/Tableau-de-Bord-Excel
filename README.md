# Tableau-de-Bord-Excel
Ce projet est un exercice pratique réalisé avec Microsoft Excel afin de mettre en pratique l’analyse et la visualisation de données à partir d’une petite base de données.
### Remarque :
Les données utilisées dans ce fichier sont fictives et servent uniquement à des fins d’apprentissage et de démonstration.
